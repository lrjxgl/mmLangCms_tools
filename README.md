# mmLangCms_tools
 mmLangCms工具,含Php\GoLang\java\uniApp等基于mysql自动生成代码工具  
# 使用方法
 config.php 配置数据库信息  
 # go 
 goControl.php go控制器生成 
 goModel.php go模型生成 
 goApi.php go生成全部  
 # php  
 phpControl.php php控制器生成  
 phpModel.php php模型生成  
 phpApi.php php生成全部  
 # java  
javaControl.php java控制器生成  
javaModel.php java模型生成  
javaApi.php java生成全部  
# uni 后台接口生成   
uniAdd.php uni发布页面  
uniList.php uni列表页  
uniApi.php uni生成全部
